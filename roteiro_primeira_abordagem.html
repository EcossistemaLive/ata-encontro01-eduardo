Roteiro de Abordagem

Roteiro — Primeira Abordagem

Script estruturado para contato com Eduardo. Objetivo: coletar os dados formais para geração do contrato e proposta comercial.

👤 Cliente: Eduardo
📲 Canal: WhatsApp
❓ Perguntas: 17
🎯 Objetivo: Dados p/ contrato
1 Mensagem de Abertura

Primeira mensagem pelo WhatsApp — tom amigável, direto e profissional.

Olá Eduardo, tudo bem? 👋

Meu nome é Cléber Donato, sou da equipe da Live Consultoria do Luiz Portal.

O Luiz me passou o briefing do seu projeto e estou muito animado com o trabalho que vamos desenvolver juntos! Já estamos com o planejamento inicial estruturado e prontos para começar.

Para darmos o próximo passo — formalizar a proposta e o contrato — preciso coletar algumas informações suas e da empresa. Posso te enviar as perguntas aqui ou prefere agendar uma call rápida de 15 minutos?

Fico no aguardo! 🚀
💡
Dica: Se ele preferir call, agende pelo Google Calendar e use as perguntas abaixo como roteiro da conversa. Se preferir texto, envie bloco a bloco para não sobrecarregar.
2 Dados da Empresa

Informações formais obrigatórias para o contrato de prestação de serviços.

📋 Bloco 1 — Dados Formais
"Primeiro, vou precisar de alguns dados formais da empresa para o contrato:"
  1. Qual é a razão social da empresa?
  2. Qual o CNPJ?
  3. Qual o endereço completo da sede?
    (Rua, número, bairro, cidade, estado, CEP)
  4. A empresa já tem nome fantasia definido?
3 Dados dos Sócios

Identificação dos responsáveis que assinarão o contrato.

👥 Bloco 2 — Representantes
"Agora preciso dos dados do representante legal que vai assinar o contrato:"
  1. Qual o seu nome completo?
  2. Qual o seu CPF?
  3. Qual o nome do outro sócio investidor?
  4. Quem será o representante legal que assinará o contrato?
    (Pode ser você ou o outro sócio)
4 Sobre o Negócio

Contexto para a proposta comercial e planejamento do Encontro 1.

🏢 Bloco 3 — Informações do Negócio
"Para eu já começar a preparar o material do primeiro encontro, preciso entender um pouco mais sobre o negócio:"
  1. Como funciona a plataforma de vocês?
    (Marketplace, SaaS, comissão por venda?)
  2. Quem são os clientes atuais?
    (Lojistas, consumidores finais, empresas?)
  3. Já possuem faturamento? Se sim, qual a faixa mensal?
  4. Qual a stack tecnológica que os programadores utilizam?
  5. A empresa já tem missão, visão e valores definidos?
5 Logística dos Encontros

Definição de agenda e formato para os 6 encontros presenciais.

📅 Bloco 4 — Agenda
"Por último, sobre a logística das nossas reuniões:"
  1. Qual o melhor dia da semana para os encontros presenciais?
  2. Qual o local preferido?
    (Sede da empresa, escritório do Luiz, coworking?)
  3. Todos os sócios poderão participar ou apenas você?
  4. Quando seria uma boa data para o primeiro encontro?
6 Fechamento

Mensagem de encerramento após o cliente responder.

Perfeito, Eduardo! 💪

Com essas informações já consigo finalizar a proposta formal e o contrato.

Vou preparar tudo e te envio para revisão antes do nosso primeiro encontro.

Qualquer dúvida, é só me chamar! 🚀
7 Follow-up (48h sem resposta)

Template para reenvio caso o cliente não responda.

Oi Eduardo! 😊

Passando para ver se conseguiu ver minha mensagem anterior.

Estamos com tudo pronto do nosso lado para iniciar o projeto — só faltam as informações da empresa para formalizarmos o contrato.

Fica à vontade para me responder quando puder! 🚀
⏰
Timing: Enviar o follow-up após 48h sem resposta. Se após mais 48h ainda não responder, alinhar com Luiz sobre próximo passo (ligar diretamente ou reagendar abordagem).
8 Checklist de Dados Coletados

Use esta lista para validar que todos os dados foram obtidos antes de gerar a proposta.

Dados Obrigatórios para Contrato

  • Razão social da empresa
  • CNPJ
  • Endereço completo (rua, nº, bairro, cidade, UF, CEP)
  • Nome fantasia
  • Nome completo do representante legal
  • CPF do representante legal
  • Nome do segundo sócio investidor

Dados para Proposta e Planejamento

  • Modelo de negócio (marketplace, SaaS, comissão)
  • Público-alvo / clientes atuais
  • Faixa de faturamento mensal
  • Stack tecnológica
  • Missão, visão e valores (se existentes)
  • Dia preferido para encontros
  • Local preferido para encontros
  • Participantes confirmados
  • Data sugerida para o Encontro 1
